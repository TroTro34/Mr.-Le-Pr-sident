\# Mr. Le Président



\## Prérequis

\- Python 3.14.2

\- pygame-ce 2.5.6



\## Installation des dépendances

pip install -r requirements.txt



\## Lancement

python sources/main.py



\## Fichiers nécessaires

Le dossier data/ doit contenir les fichiers suivants :

\- Les images des personnages (Assistant.png, Général.png, etc.)

\- Les images des boutons (vraiYES.png, vraiNON.png)

\- Les polices (Capture it.ttf, Sakire.ttf)

\- Les fichiers de dialogue (histoire.txt, question.txt, intro.txt, etc.)



\## Utilisation

Le jeu nécessite uniquement la souris durant le jeu (déplacement et clic gauche)

et la touche Échap pour fermer le jeu.



\- Lors des présentations, de l'intro et des textes spéciaux, cliquez simplement

&#x20; n'importe où sur l'écran pour avancer.

\- Pour les questions, choisissez votre réponse et cliquez sur le bouton

&#x20; correspondant (gauche = d'accord, droite = pas d'accord).

\- Échap pour quitter le jeu à tout moment.
